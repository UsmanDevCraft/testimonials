# 🌟 Testimonials

> The seamless way to collect, manage, and display social proof.

Testimonials is a premium social proof platform designed for creators and business owners. It automates the workflow of gathering, curating, and showcasing customer feedback to build trust and increase conversions.

## 🚀 How it Works

1.  **Share**: Send your unique profile link or space URL to clients.
2.  **Collect**: Clients submit text or video testimonials via a clean, immersive glassmorphism form.
3.  **Curate**: Approve and organize the best testimonials in your private, authenticated dashboard.
4.  **Display**: Integrate your "Wall of Love" onto any website using modern embeddable widgets.

## ✨ Key Frontend Features

- **💎 Premium UI/UX**: Built with a modern Dark Glassmorphism aesthetic, featuring smooth gradients and micro-animations.
- **🛡️ Secure Routing**: Authentication-guarded dashboard ensuring only you can manage your social proof.
- **📊 Dynamic Dashboards**: Real-time management of "Spaces" with TanStack Query for efficient data fetching and caching.
- **📱 Fully Responsive**: Optimized for desktop, tablet, and mobile viewing.

## 🛠 Tech Stack

| Category             | Technology                         |
| :------------------- | :--------------------------------- |
| **Frontend Core**    | React 18 (Vite)                    |
| **State Management** | TanStack Query (v5)                |
| **Routing**          | React Router DOM (v6)              |

## 🏁 Quick Start

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1.  **Clone the repository**:

    ```bash
    git clone https://github.com/UsmanDevCraft/testimonials.git
    cd testimonials
    ```

2.  **Install dependencies**:

    ```bash
    npm install
    ```

3.  **Start the development server**:
    ```bash
    npm run dev
    ```
